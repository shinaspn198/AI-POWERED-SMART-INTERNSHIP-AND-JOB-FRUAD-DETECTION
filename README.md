# 🚀 AI-Powered Internship & Job Fraud Detection System

## 📌 Overview

The AI-Powered Internship & Job Fraud Detection System is a machine learning application designed to identify fraudulent internship and job offers. It analyzes text, PDF documents, and images using AI techniques and explains the prediction with Explainable AI (XAI).

This project helps students and job seekers detect suspicious job offers before applying.

---

## ✨ Features

* ✅ Detects fraudulent job and internship postings
* ✅ Supports text, PDF, and image analysis
* ✅ OCR-based text extraction using Tesseract OCR
* ✅ Explainable AI using LIME
* ✅ RAG-based document question answering
* ✅ Interactive Streamlit web interface
* ✅ Confidence score for predictions
* ✅ User-friendly dashboard
